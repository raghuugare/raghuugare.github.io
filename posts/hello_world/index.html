Over to some code

Here’s a sample code snippet in C:

#include <stdio.h>

/* A simple function definition... */
int factorial(int n) {
    int result = 1;
    for(int i=1; i<=n; i++)
        result *= i;
    return result;
}

int main() {
    int n = 5;
    printf("Hello World! :) \n");
    printf("factorial(%d) = %d \n", n, factorial(n));
    return 0;
}

Above is a simple C-program with a function to compute the factorial of a positive integer defined & exercised.

And now, how about some Haskell ?

Here’s a tiny gem for finding the factorial of an integer n :


factorial :: Int -> Int
factorial n
    | n <= 0    = 1
    | otherwise = n * factorial (n-1)
